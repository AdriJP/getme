# getme
GETME CLI MULTITASK COMMAND v1.0

Comando creado para aportar utilidades en materia de interactuar con el sistema.
Escrito en el lenguaje de script de bash usa comandos y programas externos de la terminal
como 'whoami', 'set', 'lscpu', 'free', y 'df'. 
Su código es muy sencillo, usa un condicional para el primer argumento que se le pasa 
al comando, aparte de ciertas variables para guardar el resultado de los comandos 
y para codigos de color de bash. El codigo intenta simular cierta inspiración en 
el lenguaje de programación C, en tributo a este lenguaje y a los programas que ha 
aportado. 

Proyecto personal.
